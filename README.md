**Cross-Dataset Generalization of Underwater Instance Segmentation**

Carsten Sprunger

# **Problem & Motivation**

Marine debris threatens ocean ecosystems, and instance segmentation models can automate detection for cleanup operations. However, models trained on one underwater trash dataset may fail on data from different environments. This project systematically studies cross-dataset generalization of Mask R-CNN between two public datasets — TrashCan (TC; 7,212 images, 22 classes) and SeaClear (SC; 8,610 images, 40 classes) — which differ in camera setup, water conditions, and annotation scheme. A generic category hierarchy (TC20 → coarse → ternary → binary) was developed to enable evaluation at multiple granularity levels.

# **Approach**

We fine-tune Mask R-CNN (ResNet-50 \+ FPN, COCO-pretrained) on each of TC and SC (with various configurations) and evaluate them on *both* TC and SC via a common coarsening of the TC and SC category spaces to measure how well each generalizes cross-domain. The dataset split provided by the authors of the TrashCan dataset suffers from significant data leakage due to adjacent video frames appearing in both training and validation/test data splits, which inflates in-domain performance considerably. We re-split the data using a frame-chunking method to mitigate data leakage and obtain a fairer measure of the cross-domain generalization gaps. Additionally, we fine-tune models on datasets generated by reducing TC and SC to various common coarsenings and pooling them to measure how well these models can retain the in-domain performance of their TC-only and SC-only counterparts. Feature space analysis (t-SNE/UMAP projections, silhouette scores) characterizes the learned representations of all trained models. 

# **Key Results**

Mask mAP @ IoU 0.50 (chunksplit evaluation, evaluated in TC20 category space):

| Training Data | Evaluated on TC Test | Evaluated on SC Test |
| ----- | :---: | :---: |
| TC-only | 0.329 | 0.025 |
| SC-only | 0.021 | 0.456 |
| Pooled | 0.347 | 0.465 |

Mask mAP @ IoU 0.50 (chunksplit evaluation, trained/evaluated in coarse category space):

| Training Data | Evaluated on TC Test | Evaluated on SC Test |
| ----- | :---: | :---: |
| TC-only | 0.443 | 0.011 |
| SC-only | 0.009 | 0.518 |
| Pooled | 0.449 | 0.540 |

# **Findings**

* **Catastrophic domain gap:** Cross-dataset mAP drops 10-15x (from \~0.33-0.46 to \~0.02), confirming that single-dataset models do not generalize.

* **Pooled training recovers performance:** A single jointly-trained model matches or exceeds in-domain baselines on both test sets without architectural changes.

* **Visual, not semantic, gap:** Feature-space silhouette analysis shows strong dataset clustering (0.20-0.35) even at the per-class level, indicating the gap is driven by image-level domain differences rather than category confusion.

# **Future Work**

Domain adaptation techniques (FDA, style transfer) to close the remaining gap without pooling; attention-based architectures (e.g., Mask2Former); larger and more diverse joint datasets; deployment-oriented detection models for real-time underwater cleanup robots trained in coarsened category spaces tailored to their intended use cases.
